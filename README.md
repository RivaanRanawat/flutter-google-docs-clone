# Google Docs Clone

A completely Responsive Instagram App- Works on Android, iOS & Web! 

## Features
- Google Authentication
- State Persistence
- Creating new Documents
- Viewing List of Documents
- Updating title of Document
- Link sharing
- Auto saving
- Collaborative Editing in Rich Text Editor
- Sign Out
